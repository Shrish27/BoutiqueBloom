# Project: Local Business Marketing Support Platform

## Purpose
A web-based marketing and branding support platform for local small businesses.  
It helps business owners organize campaigns, plan social media content, explore branding packages, and manage customer communication through a clean and elegant interface.

## Target Users
- Local small business owners
- Home-based sellers
- Boutique brands
- Small cafés, shops, and service businesses

## Core Pages
1. Home page
2. Login page
3. Register page
4. Dashboard
5. Planner / Content Calendar
6. Catalogue / Services
7. Contact / Feedback

## Design Style
Atmospheric Boutique

## Color Palette
- Primary: #FDA4AF
- Secondary: #FFF7ED
- Accent: #FB923C
- Dark text: #451A03

## Typography
- Headings: Playfair Display
- Body/UI: Open Sans

## UI Feel
- Rounded cards
- Soft shadows
- Warm cream backgrounds
- Minimal elegant layout
- Pill-shaped buttons

## Notes
Frontend first.
Use static/dummy data.
No backend for now.
